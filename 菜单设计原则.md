# 菜单设计原则

生成或修改菜单时必须遵守以下规则。

## 午餐限制

- **不吃炒菜** — 午餐不安排任何炒菜（如炒鸡蛋、炒青菜等）
- **不吃白米饭** — 午餐主食按菜系选择：
  | 菜系 | 主食 |
  |------|------|
  | 中餐 | 杂粮饭 / 杂粮馒头 / 玉米 / 红薯 |
  | 日式 | 荞麦面 / 乌冬面 |
  | 韩式 | 杂粮饭 / 荞麦冷面 |
  | 东南亚 | 米粉 / 河粉 |
  | 地中海/西式 | 法棍 / 烤土豆 / 古斯古斯 |

## 菜系统一

- **每一顿饭内部菜系必须统一** — 不能在同一餐中混搭不同菜系
  - ❌ 沙葱炒鸡蛋（中国西北）+ 鹰嘴豆沙拉（地中海）
  - ❌ 白切鸡（中餐）+ 韩式豆腐汤（韩式）
  - ✅ 清蒸鲈鱼 + 凉拌海带丝 + 米饭（统一中餐）
  - ✅ 煎鸡腿排 + 鹰嘴豆沙拉（统一地中海）

## 不喜欢的食材（自动从 dislikes.json 读取）

当前列表：西兰花、全麦面包、藜麦、鸭胸

菜单中 **绝对不能出现** 这些食材。

## 灵感菜品（自动从 inspirations.json 读取）

灵感菜品应在合适的位置融入菜单，但必须遵守上述所有规则：
- 属于炒菜的灵感菜只能放晚餐
- 必须与同餐其他菜品菜系统一

## 韩式豆腐汤做法备注

正宗순두부찌개 **不用韩式辣酱（고추장）**。做法：
1. 辣椒面（고추가루）+ 蒜末热油炒出红色底油
2. 加水/高汤煮开
3. 放嫩豆腐、蛤蜊煮5分钟
4. 打入鸡蛋，出锅
